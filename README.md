# ECOS + Mami Oli's Landing (MVP)

Landing serverless en React + TypeScript + Vite para ECOS y Mami Oli's, enfocada en UX/UI premium y CTAs de conversion.

## Stack
- React + TypeScript + Vite
- Tailwind CSS
- Motion (Framer Motion)
- Embla Carousel
- Lucide React
- Vitest + Testing Library

## Scripts
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run test:run`
- `npm run lint`

## Notas
- El contenido esta centralizado en `src/content/*`.
- Las imagenes reales deben colocarse en `public/images/*`.
- El menu puede ser un PDF local (`public/documents/menu.pdf`) o URL externa configurada en `src/content/contact.ts`.
