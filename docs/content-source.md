# Content Source Guide

## Que actualizar primero
- `src/content/contact.ts`: WhatsApp, Maps, horarios, redes.
- `src/content/home.ts`: historia compartida, frases y microcopy QR.
- `src/content/gallery.ts`: rutas y textos alternativos reales.

## Imagenes reales
Colocar imagenes optimizadas en:
- `public/images/hero`
- `public/images/gallery-day`
- `public/images/gallery-night`
- `public/images/story`
